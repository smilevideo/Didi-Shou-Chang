## local testing

1. make sure ec2 instance is running/start it (needed for song upload/storage, wss can still be hosted locally)
2. `cd client`
3. `export REACT_APP_EC2_ENDPOINT=<insert EC2 url>`
4. `npm start` (in the same shell)
5. `cd server`
6. `npm start`
